SeatLock — Lifecycle
SeatLock
  │   seat            : Seat
  │   show            : Show
  │   timeoutInSeconds: int
  │   lockTime        : Date
  │   lockedBy        : User

  isLockExpired()
    └──▶ lockTime + timeout < now
         ├── false → lock still valid
         └── true  → seat free again

Timeline:
 t=0s ──────── t=600s ──────── t=∞
 LOCKED         EXPIRED
 [=============]
      ▲ 10 min window
      user must confirm + pay
Concurrency — Seat Race Condition
┌───────────────────────────────────────────┐
│  Two users fight for overlapping seats    │
└───────────────────────────────────────────┘

User1 wants seats [5, 6, 7]
User2 wants seats [7, 8, 9]  ← seat 7 overlaps!

         Thread-1          Thread-2
            │                  │
            ▼                  ▼
     lockSeats()        lockSeats()
     synchronized {     synchronized {
       check seat 7       check seat 7
       → free, lock it     → already locked!
       ✔ SUCCESS          ✘ EXCEPTION
     }                  }

  ✔ Only one thread acquires the lock.
  ✔ ConcurrentHashMap + synchronized block
     ensures atomicity.

End-to-End Booking Flow
┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│                                        END-TO-END BOOKING FLOW                                                          │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

  1. SETUP
     TheatreController.createTheatre("PVR")        ──▶ Theatre [id=1]
     TheatreController.createScreenInTheatre(...)  ──▶ Screen  [id=1]
     TheatreController.createSeatInScreen(...)  ×50 ──▶ Seat    [id=1..50]
     MovieController.createMovie("Inception", 150)  ──▶ Movie   [id=1]
     ShowController.createShow(movieId, screenId, t) ──▶ Show    [id=1]

  2. CHECK AVAILABILITY
     ShowController.getAvailableSeats(showId)
       └──▶ SeatAvailabilityService.getAvailableSeats(show)
              ├── allSeats      = screen.getSeats()          [50 seats]
              ├── bookedSeats   = bookingService.getBookedSeats(show)  [0]
              ├── lockedSeats   = seatLockProvider.getLockedSeats(show) [0]
              └── available     = allSeats - unavailable        [50 seats]

  3. CREATE BOOKING
     BookingController.createBooking(user, showId, [1,2,3])
       ├── isAnySeatAlreadyBooked?  ──▶ NO
       ├── seatLockProvider.lockSeats(show, [s1,s2,s3], user)
       │     └── synchronized { check + lock all 3 seats atomically }
       └── new Booking(id="1", show, user, seats)  status: CREATED

  4. PROCESS PAYMENT
     PaymentController.processPayment(bookingId, user)
       ├──▶ paymentStrategy.processPayment()
       │     ├── DebitCardStrategy → true  ──▶ confirmBooking()
       │     └── UpiStrategy       → false ──▶ processPaymentFailed()
       └──▶ bookingService.confirmBooking(booking, user)
              ├── user matches?          ──▶ YES
              ├── validateLock(s1,user)  ──▶ valid & not expired
              ├── validateLock(s2,user)  ──▶ valid & not expired
              ├── validateLock(s3,user)  ──▶ valid & not expired
              └── booking.confirmBooking()  status: CONFIRMED ✔

  5. RESULT
     booking.getBookingStatus()  ──▶ CONFIRMED
     getAvailableSeats(showId)   ──▶ 47 seats (seats 1,2,3 now booked)
